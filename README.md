# Proyecto Guitar 🎸

Este es un proyecto hecho con **React + Vite**.  
Se trata de una tienda de guitarras donde puedes:

- Ver una colección de guitarras.
- Agregar guitarras al carrito.
- Incrementar y disminuir cantidades.
- Guardar el carrito en LocalStorage.

## Tecnologías utilizadas
- React
- Vite
- Bootstrap
